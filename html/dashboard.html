Implemented features
This website was created as a project team work of students of the Polytechnic University who are currently undergoing distance learning course from TU Graz.
Since none of us had come across the creation of the website, we decided to use as many new technologies as we will only have time to apply.
We had to create system website with simple ajax queries to the MongoDB. As a part of this task we implemented next features:
Authentication - you may authenticate on this website using next login credentials:
login: test2@test.com
password: pass
Authorization - Authenticated users will be authorized.
Registration - You can either register your own user. But your user won't have any music available.
Profile - Each user has his own profile, that might be updated via frontend. Also we implemented feature as "team".
Music - We retrieve from MongoDB user's music and play it.
Technologies
HTML5 & JS - to form main structure and concatinate backend with frontend.
Bootstrap - framework for the layout, including wrapping containers, a powerful grid system, a flexible media object, and responsive utility classes.
MongoDB - backend database, which holds users, profiles, links, musics, and so on.
Python - backend logic written on Python is used to union the frontend and the backend. The frontend doesn't have any direct access to the MongoDB.
